What is the difference between InnerHtml, InnerText, and textContent?

innerHTML Retrieves or sets the HTML content (with tags). innerText Retrieves or sets the visible text content (without HTML tags). textContent Retrieves or sets the text content (without HTML tags, including hidden text).

What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element instead of attaching individual listeners to each child element. This allows the event handler to be triggered for events on child elements, even if they are dynamically added to the DOM later.

What is event bubbling in the DOM?

Event bubbling in the DOM is the process where an event that occurs on a child element "bubbles up" through its ancestors in the DOM tree, triggering event listeners on each of the parent elements along the way.

How do you create, add, and remove elements using JavaScript?

Create: document.createElement(tagName) creates a new element.
Add: parentElement.appendChild(newElement) adds the new element to the parent.
Remove: element.remove() or parentElement.removeChild(element) removes the element from the DOM.
